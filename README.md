dataset-xml
===========
Read Dataset-XML into a JS object
# Installation
To add dataset-xml to your project, run
```
npm install dataset-xml
```
# Usage
Read full contents of a Dataset-XML file
```
import fs from 'fs';
import { read } from 'dataset-xml';

let result = await read(pathToFile);
```
